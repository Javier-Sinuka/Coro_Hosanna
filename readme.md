# Coro Hosanna (0.1v)

Esta carpeta reúne material del cancionero del Coro Hosanna. Su objetivo es organizar canciones y referencias para distintos momentos de la celebración litúrgica, como entrada y ofertorio.

## Cancionero compilado

El cancionero se publica automáticamente como página web cada vez que se hace un push a `main`.

🔗 **[Ver Cancionero en línea](https://javier-sinuka.github.io/Coro_Hosanna/)**
